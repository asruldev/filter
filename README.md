## Kasar

**Kasar** is a Typescript or Javascript package that is used to filter profanity in paragraphs

### How to install

```bash
npm install kasar
```

### How to use

```ts
import {clearTheWords} from 'kasar'

console.log(clearTheWords('Kamu sangat goblok'))

// Kamu sangat g****k
```
